# Go Web Server

## Project Description

This is a simple web server built using the Go programming language.  
It listens on port **9000** and displays a welcome message on the homepage (`/`).

The project demonstrates how to:

- Create a basic HTTP server in Go
- Handle routes using `net/http`
- Send HTML responses to the browser

---

## Features

- Lightweight Go web server
- Handles HTTP requests
- Displays a simple HTML page
- Runs on port `9000`

---

## 🛠 Requirements

- Go installed on your system  
  Download Go from: https://go.dev

---

## How to Run

1. Clone or download the project.
2. Navigate to the project folder.
3. Run the server:

```bash
go run main.go
